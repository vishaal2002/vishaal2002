## 👋 About Me

I'm a React Native developer who thinks in terms of **mobile architecture, not just features** — I care about how apps are built underneath the screens: security posture, CI/CD ownership, and the systems that keep a codebase from turning into spaghetti as it scales. My recent focus has been mobile security tooling, static analysis, and the parts of app development that don't show up in a demo but decide whether the app survives production.

- 🔭 Currently building **audit-rn**, an AST-based static security scanner for React Native mapped to the OWASP Mobile Top 10
- 🌱 Deep in React Native architecture: component systems, deep linking, OTA pipelines, multi-environment builds
- 📫 Find me on [Medium](https://medium.com/@vishaal2002) and LinkedIn for write-ups on RN internals and security
